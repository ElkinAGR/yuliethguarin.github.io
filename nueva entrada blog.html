Presentación: Cómo Utilizar Git

Una guía paso a paso para principiantes

1. ¿Qué es Git?

Git es un sistema de control de versiones distribuido que te permite rastrear los cambios en los archivos y colaborar con otros desarrolladores. Es ideal para proyectos individuales y colaborativos.

2. Inicialización de un Repositorio Local

Para inicializar un repositorio en Git, navega a la carpeta de tu proyecto y ejecuta el siguiente comando:

git init

Esto creará una carpeta oculta llamada .git, donde se almacenarán todos los datos necesarios para el control de versiones.

3. Creación de Branches (Ramas)

Las ramas permiten trabajar en nuevas características o correcciones de errores sin afectar la rama principal (main). Para crear una rama nueva:

git branch feature-login

Cambia a la nueva rama con:

git checkout feature-login

4. Fusionar Ramas (Merge)

Una vez que termines de trabajar en una rama, puedes fusionarla con la rama principal:

git checkout main
git merge feature-login

Esto integrará los cambios de la rama feature-login en la rama principal main.

5. Conectar a un Repositorio Remoto

Para conectar tu repositorio local a GitHub, utiliza el siguiente comando:

git remote add origin https://github.com/tu-usuario/mi-proyecto.git

Sube los archivos al repositorio remoto con:

git push -u origin main

¡Gracias por ver la presentación!

Ahora sabes cómo inicializar un repositorio, crear ramas, fusionarlas y conectar tu proyecto a un repositorio remoto en GitHub.
